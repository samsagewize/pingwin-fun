# PINGWIN.FUN

Meme-casino launchpad (UI-first scaffold).

## Stack
- Next.js (App Router) + TypeScript
- Tailwind CSS v4 (CSS-first theme tokens in `src/app/globals.css`)

## Dev

```bash
npm install
npm run dev
```

Open http://localhost:3000
